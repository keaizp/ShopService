## 四、响应格式检查

### 4.1 成功响应
```json
{
    "code": 200,
    "message": "success",
    "data": {}
}
```
- [ ] code为200
- [ ] message为"success"
- [ ] data不为null（空数组或对象）

### 4.2 错误响应
```json
{
    "code": 400/500,
    "message": "错误信息",
    "data": null
}
```
- [ ] 错误码正确
- [ ] 错误信息清晰
- [ ] data为null

### 4.3 分页响应
```json
{
    "code": 200,
    "message": "success",
    "data": {
        "records": [],
        "total": 0,
        "size": 10,
        "current": 1,
        "pages": 0
    }
}
```
- [ ] records为数组
- [ ] total正确
- [ ] size/current/pages正确
